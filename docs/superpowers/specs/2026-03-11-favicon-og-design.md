# Favicon & OG Image Design Spec

**Date:** 2026-03-11
**Status:** Approved

## Summary

Add a favicon (SVG + ICO fallback) and Open Graph image (1200×630 PNG) to the Crayola Colors app. Design direction: **Color Grid** — a 2×2 mosaic of 4 iconic Crayola colors.

## Favicon

**File:** `public/favicon.svg` (primary), `public/favicon.ico` (16/32px fallback)

A 2×2 grid of four color cells, no gap between cells, rounded outer corners.

| Cell | Color | Hex |
|---|---|---|
| Top-left | Red | `#ED0A3F` |
| Top-right | Blue | `#1F75FE` |
| Bottom-left | Yellow | `#FDDB6D` |
| Bottom-right | Green | `#1CAC78` |

SVG viewBox: `0 0 32 32`. Corner radius: 6px on outer corners only (clip-path or rx on wrapping rect).

`index.html` link tags:
```html
<link rel="icon" type="image/svg+xml" href="/colors/favicon.svg">
<link rel="icon" type="image/x-icon" href="/colors/favicon.ico">
<link rel="apple-touch-icon" href="/colors/apple-touch-icon.png">
```

## OG Image

**File:** `public/og-image.png` — 1200×630px, generated by screenshotting an HTML template

**Layout:** Two-panel horizontal split

- **Left panel (55%):** White background
  - Title: "Crayola Colors" — bold, large, dark (`#111`)
  - Subtitle: "242 colors · full history" — muted (`#999`)
- **Right panel (45%):** 3×4 grid of 12 color swatches (no gaps, fills the full panel height)

**12 mosaic colors** — curated rainbow spread:
`#ED0A3F`, `#FF7538`, `#FDDB6D`, `#1CAC78`, `#1F75FE`, `#7442C8`, `#FFCBA4`, `#FC2847`, `#000000`, `#8E4585`, `#29ABCA`, `#FFFFFF`

## `<meta>` Tags

Add to `index.html`:
```html
<meta property="og:title" content="Crayola Colors">
<meta property="og:description" content="Browse 242 Crayola crayon colors with full production history, name changes, and variants.">
<meta property="og:image" content="https://marchdoe.github.io/colors/og-image.png">
<meta property="og:url" content="https://marchdoe.github.io/colors/">
<meta property="og:type" content="website">
<meta name="twitter:card" content="summary_large_image">
```
